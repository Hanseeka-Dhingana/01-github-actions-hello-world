# 👋 Hello World GitHub Action

Welcome! This repository shows a **basic GitHub Actions workflow** in action 


##  What happens here?
✅ Workflow runs automatically  
✅ Triggered on **push to `main`**  
✅ Uses **Ubuntu runner**  
✅ Prints a **Hello World** message  



## How does it run?
1. You push code to `main`
2. GitHub detects the push 
3. The action runs automatically 
4. You see the output in the **Actions** tab 



## Why this repo?
Perfect for:
- Beginners learning GitHub Actions
- Understanding CI/CD basics
- Testing automation setup

Simple. Automatic. Powerful.
